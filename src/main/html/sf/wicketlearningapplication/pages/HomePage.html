Wicket Learning Application


This application demonstrates:
  • Login (and authentication)
  • Logout
  • Navigation from one page to another
  • Paged lists of domain objects
  • Forms that allow adding domain objects
  • Drop-down lists in forms
  • JavaScript "behaviors", such as date pickers
  • Validation of input
  • Display of error messages
  • Custom style sheets
  • Mounting URLs
  • Dynamically generated JavaScript
  • Automatic binding of properties to elements with wicket ids that are the same as the domain object properties
  • Localization of messages
